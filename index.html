投資策略分析

策略介紹

它是一個「區間突破」系統。核心觀念是利用前 N 日的最高價 (High)、最低價 (Low) 和收盤價 (Close) 來決定震盪區間 (Range)。

核心公式Range = Max(HH - LC, HC - LL)

HH = 前 N 日最高價的最高值
LC = 前 N 日收盤價的最低值
HC = 前 N 日收盤價的最高值
LL = 前 N 日最低價的最低值

為什麼 Dual Thrust 這麼強？

用前 N 天的價格真實波動幅度，畫出一個動態箱體，今天只要突破箱體就視為「市場真的動了」，因此進場跟趨勢。
